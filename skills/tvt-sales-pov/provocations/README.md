# Stage 1 provocations

3-5 hand-curated, mortgage/fintech-specific provocations for the POV-led discovery motion (spec 006
§3 build-sequencing). Each provocation should name a problem the target hasn't yet articulated
themselves, backed by a quantifiable impact claim.

Format per provocation: `problem_we_see.md`, `impact_we_anticipate.md`, `change_we_propose.md` — or
a single `<slug>.md` with those three headers. Example seed: "AI underwriting fails at
self-employed borrowers."

Empty until the first provocations are written — this is Stage 1 content work, not a code
dependency, and can start immediately without waiting on Salesforce/Notion MCP access.
